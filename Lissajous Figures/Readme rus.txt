Со временем точность вычислений и, как следствие, качество отрисовки, падает. Поэтому после 5 тысяч отрисованных
периодов функции счетчик периодов сбрасывается. Этот эффект особенно заметен при...
Freq X = 40
Freq Y = 60
...особенно, если растянуть амплитуду по X на всю ширину окна.

Кстати говоря, во время разработки случайненько обнаружил, что при...
Freq X = 19.99
Freq Y = 59.95
...анимация в 5 тысяч периодов красиво замыкается и разрыва кадров при переходе с 5-тысячного значения на
нулевое не происходит (либо он практически не заметен).

Управление:
Up / Down - медленное изменение частоты по Y
Right / Left - медленное изменение частоты по X

Ctrl + (Up / Down) - быстрое изменение частоты по Y
Ctrl + (Right / Left) - быстрое изменение частоты по X

Shift + (Up / Down) - медленное изменение амплитуды по Y
Shift + (Right / Left) - медленное изменение амплитуды по X

Shift + Ctrl + (Up / Down) - быстрое изменение амплитуды по Y
Shift + Ctrl + (Right / Left) - быстрое изменение амплитуды по X
